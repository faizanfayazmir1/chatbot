# Number Color Chatbot

A simple frontend chatbot that colors acknowledgments based on the last two digits of a user-entered number.

## Features

- Input a number
- Acknowledgment colored from white (00) to black (100), with grey at 50
- WCAG 2.0 compliant (keyboard accessibility, contrast ratios)

## Deployment

To deploy on Vercel:

1. Install Vercel CLI: `npm install -g vercel`
2. In the project directory: `vercel`
3. Follow the prompts to deploy.

Or, drag the folder to [vercel.com](https://vercel.com) for quick deployment.

## Local Development

Open `index.html` in a browser.
